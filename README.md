Este proyecto es un sistema de punto de venta (POS) desarrollado en Node.js, diseñado para la gestión de ventas e inventario en tiendas comerciales. Permite a los usuarios realizar compras de manera rápida y eficiente, mientras que los administradores pueden supervisar y controlar la operación del negocio.

-Características principales
Sistema de autenticación por roles
Administrador
Cajero
Usuarios no registrados (compra libre)

Módulo de ventas
Registro de productos
Cálculo automático de precios
Generación de tickets de compra
 Gestión de inventario
Control de stock
Actualización automática tras cada venta
Envío de tickets por correo electrónico
Generación de comprobantes digitales
Envío automático al cliente
Descargas de reportes mensual, quincenal o personalizado
- Arquitectura
Basado en modelo cliente-servidor
Backend desarrollado con Node.js
-Tecnologías utilizadas
Node.js
Xampp

Se desarrolló una solución funcional para la administración de ventas e inventario en pequeños y medianos negocios, simulando el funcionamiento de sistemas comerciales como los utilizados en supermercados.
